# Web Portfolio

Este es un portafolio web desarrollado para mostrar los servicios y habilidades de un programador frontend.

**Ezequiel Brito Navarro**  
Email: ebritonavarro@gmail.com  

## Descripción

El portafolio incluye una sección de servicios, proyectos destacados y un formulario de contacto. Está diseñado con un enfoque en la experiencia de usuario (UX) y la interfaz de usuario (UI), utilizando tecnologías modernas.

## Tecnologías utilizadas

- **React**: Biblioteca para construir interfaces de usuario.
- **Sass**: Pre-procesador CSS para estilos más organizados.
- **React Awesome Reveal**: Animaciones para mejorar la experiencia visual.
- **Material-UI Icons**: Iconos modernos y personalizables.

## Características principales

- **Servicios**: Presentación de los servicios ofrecidos, como diseño UI/UX, mantenimiento, integración de APIs, optimización de rendimiento y desarrollo de aplicaciones móviles.
- **Proyectos**: Muestra de proyectos destacados con descripciones y enlaces.
- **Responsive Design**: Diseño adaptable para dispositivos móviles y de escritorio.
- **Formulario de Contacto**: Permite a los usuarios enviar mensajes directamente.

## Instalación y ejecución

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/portfolio.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd portfolio
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Ejecuta el proyecto en modo desarrollo:
   ```bash
   npm start
   ```
5. Abre tu navegador en `http://localhost:3000`.
